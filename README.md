# Calculator Scrips
Python programs written for the Python Edition of the Ti-84 Plus CE

## List of programs
### Fun
#### BALL
0's go back and forth across the screen indefinetly

### Math
#### ANGLES
Convert angle measures between radians and degrees.
#### COSINES
Find missing angles and sides of a triangle using Law of Cosines.
#### EULER
Find approximations of points of a function using Euler's Method when given a dervative who's anti-dervative cannot be found by using seperation of variables.
#### FACTORS
Prints all the factors of a number.
